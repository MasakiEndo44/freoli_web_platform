## Gate 1（技術的適合性）— 全 PR 必須

- [ ] 認可（IDOR 防御）：ユーザースコープが全 CRUD で確認されている

- [ ] 型安全境界：API 入力値が Zod 等で検証されている
- [ ] N+1 / 漏洩クエリ：ループ内 DB 呼び出しなし / `select('*')` で不要カラム露出なし
- [ ] エラー境界：内部エラーオブジェクトをそのままクライアントに返していない
